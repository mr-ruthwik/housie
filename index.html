How to use this website:

  • Enter your name and choose number of tickets (max 4).
  • Create Room to become host and share Room Code.
  • Join Room using code from host.
  • Host uses Generate Number button.
  • Tap numbers on ticket when they appear.
  • Claim patterns like Jaldi, Lines, or Housie.
  • Leave room anytime using Leave Room button.

How to play the Housie game:

  • Jaldi 5: First 5 numbers.
  • Lines: Complete any row.
  • Full House: All 15 numbers.
  • Validation: System checks your claim.
